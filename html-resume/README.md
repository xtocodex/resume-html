# Single-Page Resume Website

A clean, semantic HTML only resume website built without any CSS styling....

## Project Overview

This is a single-page resume website created using only HTML without any CSS. It includes all essential sections of a professional resume:
- Header with name and professional title
- About Me section
- Skills list
- Experience table
- Projects showcase
- Education table
- Contact information
- Footer

##  Features

- **Semantic HTML5**: Used proper semantic tags 
- **Accessible Tables**: Well-structured tables for experience and education
- **Clean Structure**: Organized sections with clear hierarchy
- **Responsive**: Basic HTML structure that works on all devices
- **No Dependencies**: Pure HTML - no CSS, JavaScript, or external libraries

##  Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- A text editor (VS Code, Notepad++, etc.)

### Installation Steps

1. **Clone or Download the Repository**
   ```bash
   git clone <github.com/xtocodex/html-resume>
   cd html-resume
   ```

2. **Open the HTML File**
   - Simply double-click on `resume.html` to open it in your default browser
   - Or right-click → "Open with" → Choose your preferred browser

### Alternative: View Locally
```bash
# If you have Python installed or VSCODE LIVESEVER, you can run a local server:
python -m http.server 8000
VSCODE - Live Server Extension  {INSTALL AND RUN}

# Then visit: http://localhost:8000/resume.html
```

## Usage

### Viewing the Resume
1. Open `resume.html` in any web browser
2. The page will display the complete resume with all sections
